# Build Own Front

<Callout type="warning" emoji="⚠️">
  This guide assumes that you have experience working with `React` and `GraphQL`.
</Callout>

Azuro protocol logic is laid down in smart contracts. [Read more about Azuro contracts](/dive-deeper/contracts).

To simplify client side development Azuro uses [Subgraphs](https://thegraph.com/docs/en/about/). The result of this is
GraphQL that can be used to fetch data from blockchain. [Read more about Azuro GraphQL](/dive-deeper/graphql).
