# Redeem Bet

```ts
withdrawPayout(
  core: PromiseOrValue<string>,
  tokenId: PromiseOrValue<BigNumberish>,
  isNative: PromiseOrValue<boolean>,
  overrides?: Overrides & { from?: PromiseOrValue<string> }
): Promise<ContractTransaction>;
```

```ts
// in Gnosis there is xDAI (native) token used for bets
const tx = await lpContract.connect(signer).withdrawPayout(coreAddress, bet.betId, true)

// in Polygon there is USDC (non native) token used for bets
const tx = await lpContract.connect(signer).withdrawPayout(coreAddress, bet.betId, false)
```
